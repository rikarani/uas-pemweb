# Ritual

1. clone repo

```
git clone https://github.com/rikarani/belajar-laravel
```

<br>

2. copas ini

```
composer install
```

<br>

3. trus copas ini

```
npm install
```

<br>

4. Copy file .env.example

```
copy .env.example .env
```

<br>

5. Generate App Key

```
php artisan key:generate
```

<br>

6. Jalanin Servernya

```
php artisan serve
```

<br>

7. Jalanin HMR

```
npm run dev
```
